Quadro comparativo de linguagens:

Linguagem Características Soluções que utilizam
JavaScript Orientada a objetos, executado no browser, independe de plataforma, cria funcionalidades para websites. Netflix, Uber, Amazon, Facebook, Airbnb.
Java Orientada a objetos, tipada, compilada, múltiplas aplicabilidades. Uber, Glovo, Spotify, BlaBlaCar.
PHP Pode ser orientada a objetos, integração com múltiplos banco de dados, código aberto. Facebook, Lyft, Tesla, BlaBlaCar.
Phyton Código aberto, multiplataforma, tipagem dinâmica, orientada a objetos. Netflix, Spotify, Google, Instagram, Uber, NASA.
C# Orientado a objetos, multiplataforma, fortemente tipada. Delivery Hero, Stack Overflow, Clear Bank.
SQL Gerenciador de banco de dados, pode ser orientado a objetos. Vivo, Itaú, Amil.
